== Description ==

  This plug-in is used to display testimonials. 

- It is compatible for any WordPress site.
- Lists testimonials same as WordPress post / page lists.
- Also lists categories for testimonial.
- Manage testimonial title, description, position, etc.
- Generates dynamic short-code for testimonials.
- It is managed with three different layouts : Slider, Grid and List layout.
- It is displayed with different 5 styles.
- Also displayed based on ordering and display with dynamic font size, language and color as well as image size and radius.

== Installation ==

1. Upload the entire esb-testimonials folder to the /wp-content/plugins/ directory, or just upload the ZIP package via 'Plugins > Add New > Upload' in your WP Admin
2. Activate ESB Testimonials from the 'Plugins' page in WordPress
